# Scoopify 🍦
Scoopify is a simple ice‑cream ordering web application where users can:
- Browse ice creams
- Choose toppings
- Add items to cart
- Add items to wishlist
- Place orders
- View order history
- Register and login securely with strong password validation

The application also includes an Admin Panel where administrators can:
- Add new ice‑cream products
- Manage or delete products
- View customer orders

Built using React + Tailwind CSS with a Node.js, Express, and MongoDB backend.
